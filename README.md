# 基于Uniapp的新闻网

# 一：技术工具环境
      技术：Vue、Uniapp、Html5、CSS3、Js、Jq   
      工具：HBuilderX   
      环境：Chrome   
# 二：项目功能
首页模块、详细页模块、个人中心模块   
    
      首页模块：分类导航栏、浏览量、新闻列表  
      详细页模块：编辑人、发布日期、声明、新闻图片内容展示  
      个人中心模块：浏览历史
